# Krynn Tools — Tool Taxonomy & Catalog

> Complete category taxonomy and URL mapping for 130+ free tools.

## Categories Overview

1. **PDF Tools** (`/pdf`): Compress, merge, split, convert, crop, sign, watermark, and OCR PDF files completely client-side using `pdf-lib` and `pdf.js`.
2. **Image Tools** (`/image`): Compress, resize, crop, rotate, convert (PNG, JPG, WebP, SVG, HEIC, TIFF, RAW, PSD), remove background, and watermark images via HTML5 Canvas.
3. **Text Tools** (`/text`): Word counter, case converter, line sorter, duplicate remover, text diff checker, HTML encoder, Markdown previewer.
4. **Developer Tools** (`/dev`): JSON formatter & validator, Base64 encoder/decoder, URL encoder, UUID generator, Regex tester, JWT decoder, SQL formatter.
5. **Web Design Tools** (`/design`): CSS gradient generator, box shadow builder, border radius designer, color palette tool, font pairer, SVG wave generator.
6. **Calculators** (`/calculators`): BMI calculator, loan EMI, age, percentage, compound interest, discount, GST/VAT, currency, unit converter.
7. **Security Tools** (`/security`): Password generator, strength analyzer, MD5/SHA256 hash generator, QR code creator & scanner.
8. **Misc Tools** (`/misc`): Barcode generator, random number, dice roller, coin flip, timezone converter, file checksum calculator.
9. **Social Media Tools** (`/social-media`): Instagram bio counter, username validator, social media image resizer, hashtag counter.
10. **AI Writing Tools** (`/ai-writing`): Essay writer, blog post generator, paragraph writer, content improver, grammar fixer, powered by serverless Gemini API.
