## About Krynn Tools
Krynn Tools is a privacy-focused suite of 130+ free web utilities operating across PDF management, image manipulation, developer formatting, web design, calculators, security generators, and AI writing assistants.

## Key Principles & Architectural Guarantees
1. **100% Client-Side Execution**: All processing for PDF, Image, Text, Dev, Design, Security, and Calculators occurs strictly within the user's browser using Canvas API, Web APIs, and WebAssembly.
2. **Zero Data Uploads**: User files, text inputs, and generated data never touch remote servers.
3. **No Auth Friction**: Zero accounts, zero login walls, zero paywalls.
4. **Free Forever**: Completely free for personal and commercial usage.
